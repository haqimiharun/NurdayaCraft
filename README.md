# NurdayaCraft

A PHP e-commerce web application for handmade craft products.

## Features

- User authentication (login/register)
- Product catalog with categories
- Shopping cart and checkout
- Order management
- Admin dashboard

## Tech Stack

- PHP (MVC architecture)
- MySQL database
- HTML/CSS/JavaScript

## Setup

1. Clone the repository
2. Copy `.env.example` to `.env` and configure your database credentials
3. Import `nurdaya.sql` to your MySQL database
4. Configure your web server to point to the project directory
5. Access via `http://localhost/NurdayaCraft`

## Project Structure

```
├── assets/         # CSS, JS, images
├── config/         # Configuration files
├── controllers/    # MVC controllers
├── core/           # Core framework classes
├── uploads/        # Product images
└── views/          # PHP view templates
```
